# Crypto Suite 🔐

Crypto Suite is an interactive web application that helps users explore and understand fundamental cryptography concepts through hands-on tools.

The application provides simple tools for experimenting with classical and modern cryptographic techniques.

## Features

* Morse Code Encoder / Decoder
* Caesar Cipher Encryption & Decryption
* RSA Encryption Demonstration
* Interactive UI for experimenting with cryptography
* Clean and responsive interface

## Tech Stack

* React
* TypeScript
* Vite
* Tailwind CSS
* shadcn/ui

## Installation

Clone the repository:

git clone https://github.com/Lokesh-up/crypto-suite.git

Navigate to the project directory:

cd crypto-suite

Install dependencies:

npm install

Run the development server:

npm run dev

The application will start at:

http://localhost:5173
